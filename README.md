# Zecpath AI System

## Project Structure

- data/ → Stores raw and processed data
- parsers/ → Resume parsing modules
- ats_engine/ → Resume scoring logic
- screening_ai/ → AI call & screening logic
- interview_ai/ → HR & Technical interview engines
- scoring/ → Score aggregation logic
- utils/ → Logger and helper functions
- tests/ → Unit tests

## Setup Instructions

1. Create virtual environment
2. Activate venv
3. Install requirements
4. Run main.py
5. Run pytest for testing
